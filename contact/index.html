PH-Tools

PH-Tools is the umbrella project for all the software tools we've been building to make Passive House energy modeling faster, easier, and more fun. The Honeybee-PH plugin is our newest toolkit, written for Honeybee from the ground up to work seamlessly with Ladybug-Tools.

Get Help

Having trouble and can't find the answer? See How to Ask for Help below.

  • Check the Resources page for detailed how-to guides and tutorials.
  • Installation issues? Check the Install page.
  • Rhino or Grasshopper trouble? Check the Rhino Forum.
  • Ladybug-Tools or Honeybee trouble? Check the Ladybug-Tools Forum.
  • Can't find the answer? Try posting on the Ladybug-Tools Forum. Public questions become a resource for future users.
  • Found a bug? Open a GitHub Issue.
  • Still stuck? Email us at: phtools@bldgtyp.com

How to Ask for Help

We're happy to help, but we need enough information to understand what's happening. Please include the following in any help request or forum post:

Current Behavior / Issue

What is the problem? What scenario are you having trouble with? Be as specific as possible.

Expected Behavior

What did you expect to happen?

What You've Tried

What steps have you already taken to solve the problem?

Environment Details

  • Operating System and version (Windows, Mac, etc.)
  • Rhino version
  • Ladybug Tools version
  • PHPP version and language (if exporting to PHPP)
  • WUFI-Passive version (if exporting to WUFI)
  • Versions of any other relevant plugins

Screenshots / Examples

We can't see your computer — provide screenshots or example files so we can understand what's happening.

Example Files

When providing sample files, remember:

  • We don't have access to your computer or network — ensure any linked files are internalized to the GH file.
  • For weather files, use the LBT download weather file tool — hard-coded paths will cause errors.
  • Geometry doesn't travel with the GH file automatically — pipelines won't work for the receiver.
  • We don't have the same GH plugins — remove anything other than vanilla GH, Honeybee, or Honeybee-PH components before sharing.
  • Distill the GH definition down to the bare minimum to isolate the issue. Often this reveals the problem.
